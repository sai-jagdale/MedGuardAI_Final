## 🚀 About This Project

MedGuard AI is an intelligent **medicine verification system** that detects counterfeit, expired, or unsafe drugs using AI.

> 🔥 This is **Phase 2** of the project
> 🔗 Phase 1: [https://github.com/sai-jagdale/MedGuard-AI](https://github.com/sai-jagdale/MedGuard-AI)

Phase 2 introduces:

* Multi-modal verification (Text, Image, Barcode)
* AI Assistant (Gemini API)
* Advanced backend + database integration
* Full history tracking system

---

## 🎯 Features

* 🔍 Medicine Verification (Text / Image / Barcode)
* 🧠 AI Detection System
* 🤖 AI Health Assistant (Home remedies + suggestions)
* 🗂️ History Module (All 4 input types)
* ⚡ Fast API-based architecture

---

## 🛠️ Tech Stack

**Frontend:** React, HTML, Tailwind CSS, JavaScript
**Backend:** Python, Flask
**Database:** PostgreSQL + pgvector
**AI & APIs:**

* Google Vision API
* Gemini API
* Google Custom Search

---

## 🧩 Workflow

User Input → React UI → Django REST Framework Backend → Processing Layer
→ OCR / Barcode → AI Verification → Database
→ Gemini AI Assistant → Output + History Storage
