## 📌 Project Overview

**MedGuard AI** is an AI-powered medical assistant system designed to provide medicine verification, symptom-based analysis, and intelligent medical information retrieval using RAG (Retrieval-Augmented Generation).

### The system integrates:

- Secure Backend APIs  
- AI/ML-powered semantic search  
- FAISS vector database  
- PostgreSQL database  
- Web + Mobile Application  
- Dockerized deployment
